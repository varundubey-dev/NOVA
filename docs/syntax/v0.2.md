# NOVA Language Specification

### Version 0.2

NOVA v0.2 expands upon v0.1 by introducing:

* Booleans
* Any Type (`U`)
* Constants
* Null Values
* Comparison Operators
* Logical Operators

---

# Features

NOVA v0.2 supports:

* Numbers
* Strings
* Booleans
* Any Type
* Variables
* Constants
* Arithmetic Expressions
* Comparison Expressions
* Logical Expressions
* Null Values
* Output
* Comments

---

# Primitive Types

| Type | Description |
| ---- | ----------- |
| `S`  | String      |
| `N`  | Number      |
| `B`  | Boolean     |
| `U`  | Any Type    |

Examples:

```text
name: S = "Varun"

age: N = 19

isCoder: B = true

anything: U = "Hello"
anything = 10
anything = false
```

---

# Null Values

`null` is a built-in value representing the absence of data.

Examples:

```text
name: S = null

age: N = null

anything: U = null
```

Checking for null:

```text
value == null
```

Invalid:

```text
null + 5
```

Error:

```text
Cannot perform arithmetic on null.
```

---

# Variable Declaration

Variables are mutable by default.

```text
name: S = "Varun"

age: N = 19

isCoder: B = true
```

---

# Constants

Constants are immutable variables.

Use a double colon (`::`) to declare a constant.

```text
PI:: N = 3.14159

APP_NAME:: S = "NOVA"
```

Deferred initialization is allowed.

```text
VERSION:: S

VERSION = "1.0"
```

Invalid:

```text
VERSION = "2.0"
```

Error:

```text
Cannot reassign immutable variable.
```

---

# Assignment

Variables may be reassigned.

```text
name: S = "Varun"

name = "Nova"
```

Datatype rules still apply.

```text
name = 100
```

Error:

```text
Datatype mismatch.
```

# Arithmetic Operators

| Operator | Purpose        |
| -------- | -------------- |
| `+`      | Addition       |
| `-`      | Subtraction    |
| `*`      | Multiplication |
| `/`      | Division       |
| `%`      | Modulus        |

Examples:

```text
5 + 3

10 - 2

4 * 5

20 / 4

10 % 3
```

---

# Comparison Operators

| Operator |
| -------- |
| `==`     |
| `!=`     |
| `<`      |
| `>`      |
| `<=`     |
| `>=`     |

Example:

```text
age >= 18
```

Returns:

```text
true
```

or

```text
false
```

---

# Equality Rules

Numeric values are compared by value.

```text
5 == 5
```

Result:

```text
true
```

```text
5 == 5.0
```

Result:

```text
true
```

Different datatypes are never equal.

```text
5 == "5"
```

Result:

```text
false
```

---

# Logical Operators

| Operator | Meaning | 
| -------- | ------- |
| `&&`     | AND     | 
| `||`     | OR      |
| `!`      | NOT     |   

Examples:

```text
age >= 18 && isCoder

loggedIn || isAdmin

!isBlocked
```

# Comments

## Single-Line Comments

```text
// This is a comment
```

## Multi-Line Comments

```text
/ *
This is a comment
spanning multiple lines
* /
```

---

# Program Execution

NOVA executes statements from top to bottom.

```text
name: S = "Varun"

age: N = 19

isAdult: B = age >= 18

print(isAdult)
```
