# NOVA Language Specification

### Version 0.6

NOVA v0.6 expands upon v0.5 by introducing:

* While Loops
* Range Loops
* Array Iteration
* Loop Scope

---

# New Features

## While Loop

A while loop repeatedly executes a block while its condition evaluates to `true`.

```text
i: N = 0

while i < 5 {
    print(i)
    i += 1
}
```

Output:

```text
0
1
2
3
4
```

---

## While Loop (False Condition)

```text
i: N = 10

while i < 5 {
    print(i)
}
```

Output:

```text
(no output)
```

---

## Range Loop

### Exclusive Range

```text
for i in 0..5 {
    print(i)
}
```

Output:

```text
0
1
2
3
4
```

---

### Inclusive Range

```text
for i in 0...5 {
    print(i)
}
```

Output:

```text
0
1
2
3
4
5
```

---

## Descending Ranges

```text
for i in 5..0 {
    print(i)
}
```

Output:

```text
5
4
3
2
1
```

---

### Inclusive Descending Range

```text
for i in 5...0 {
    print(i)
}
```

Output:

```text
5
4
3
2
1
0
```

## Iterable Types

The following values are iterable:

* Arrays
* Nested Arrays
* Ranges

## Array Iteration

Arrays may be iterated directly.

```text
names: [S] = [
    "Varun",
    "Alex",
    "John"
]

for name in names {
    print(name)
}
```

Output:

```text
Varun
Alex
John
```

---

## Nested Loops

```text
for i in 0..3 {

    for j in 0..3 {
        print(i, j)
    }

}
```

Output:

```text
0 0
0 1
0 2

1 0
1 1
1 2

2 0
2 1
2 2
```

---

## Loop Scope

Variables declared inside a loop only exist within that loop.

```text
while true {

    temp: N = 10

}
```

Invalid:

```text
print(temp)
```

Error:

```text
Variable 'temp' does not exist.
```

---

## Loop Variables

Loop variables are automatically created by the loop.

```text
for i in 0..10 {
    print(i)
}
```

`i` exists only within the loop body.

Invalid:

```text
print(i)
```

Error:

```text
Variable 'i' does not exist.
```

---

## Infinite Loops

The following loop is valid:

```text
while true {
    print("Running")
}
```

Programs may terminate it manually or through runtime interruption.

---

## Boolean Conditions

While loops require a boolean condition.

Valid:

```text
while age < 18 {
}
```

```text
while isRunning {
}
```

Invalid:

```text
while 10 {
}
```

Error:

```text
Condition must evaluate to a Boolean value.
```
