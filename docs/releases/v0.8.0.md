# NOVA v0.8.0

The eighth public release of NOVA.

NOVA v0.8 introduces NOVA's first standard library, providing a collection of built-in functions for common programming tasks.

Programs can now manipulate arrays and strings, perform mathematical operations, convert between primitive datatypes, and read user input without requiring user-defined implementations. These built-in functions integrate seamlessly with NOVA's existing type system, function call semantics, and runtime diagnostics.

---

## Highlights

* Array functions
* String functions
* Type conversion functions
* Mathematical functions
* Input functions
* Built-in function dispatch
* Built-in function type validation
* Built-in function argument validation
* Playground-ready input abstraction

## Next Release

Planned for v0.9:

* Module imports
* Multi-file programs
* Module resolution
* Exported declarations
* Standard library imports
* Circular import detection

The next release will introduce NOVA's module system, allowing programs to be organized across multiple source files through explicit imports while maintaining predictable name resolution, lexical scoping, and compile-time diagnostics.
